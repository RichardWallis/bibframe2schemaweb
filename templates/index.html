Bibframe2Schema.org is a community initative with the following initial objectives:

  • The creation of a reference mapping from BIBFRAME 2.0† to Schema.org.
  • The development and sharing of reference software implementation(s) to:
    • Enrich BIBFRAME data with Schema.org Terms
    • Create Schema.org terms from BIBFRAME 2.0 data

†Bibframe 2.0 as per the Library of Congress BIBFRAME definition and MARC conversion specifications.

To assist understanding and development this site hosts a Comparison Tool. This web tool enables the loading of Bibframe files, including those available from the Library of Congress, and proccessing them with the latest version of conversion script, demonstrating the Schema.org output equivalent.

This site links to the W3C Community Group that supports the discssions towards the evolution of relevant tools and conversion standards.

Discussions take place on the community mailing list, and within the community wiki. These are open to all to participate.

Produced code and conversion scripts are held in a Github repository under an open source license.

For questions and furher details email info@bibframe2schema.oeg, or contact Community Group Chair Richard Wallis - @rjw.
